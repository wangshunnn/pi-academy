# 🏰 Pi Academy — 终端闯关游戏

在 Pi Coding Agent 里通过闯关学习 Pi 框架核心概念。

## 安装

```bash
pi install git:github.com/YOUR_USERNAME/pi-academy
```

## 使用

安装后在 pi 中输入：

```
/academy
```

## 学习路径

建议先通基础之路（Lv.0-8），再挑战进阶熔炉（Lv.9-Boss2）。

基础之路教你 **怎么用** — API 的使用方法。
进阶熔炉教你 **为什么这样设计** — 背后的架构思想与算法。
